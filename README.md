# ai-chatbot
Project Overview

This project implements a simple AI-based chatbot using Python. The chatbot can interact with users by responding to predefined inputs and demonstrates the basic working of conversational AI.
It is designed as a beginner-friendly project to understand chatbot logic, text processing, and user interaction.
Objective

Technologies Used

Python

Jupyter Notebook

NLTK / basic text processing (if used)

Standard Python libraries
Features of the Chatbot

Accepts user input from the console

Matches input with predefined patterns

Generates suitable responses

Runs continuously until the user exits

Easy to modify and extend responses
How the Chatbot Works

The user enters a message

The chatbot processes the input

Input is compared with predefined keywords or patterns

A matching response is returned

The conversation continues until an exit command is given

How to Run the Project

Download or clone the project

Open the notebook file:

ai chatbox.ipynb


Run all cells in Jupyter Notebook or Google Colab

Start chatting with the chatbot by entering text input
